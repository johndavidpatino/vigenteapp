# Senior Connexion UI System

Prototipo inicial HTML/CSS para capturar el estilo visual de Vigente App antes de decidir la tecnologia movil definitiva.

## Objetivo

La interfaz debe sentirse calida, humana, clara, confiable y facil de usar para adultos senior.

## Estructura

- `index.html`: pantalla demo de inicio.
- `src/styles/tokens.css`: colores, tipografia, espacios, radios y sombras.
- `src/styles/base.css`: estilos base y layout general.
- `src/styles/components.css`: componentes reutilizables.
- `AGENTS.md`: reglas para asistentes de codigo y colaboradores.

## Componentes actuales

- Header
- Logo
- Icon button
- Card
- Button
- Hero agent card
- Feature grid
- Weather card
- Recommendation card
- Reminder row
- Bottom navigation
- Agent card base

## Reglas

- No usar estilos inline.
- No inventar colores fuera de `tokens.css`.
- Mantener textos en espanol.
- Mantener layout mobile-first.
- Usar botones grandes y tarjetas legibles.
- Mantener tono humano, no robotico.
